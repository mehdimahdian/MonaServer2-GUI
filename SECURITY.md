# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| Latest release | ✅ |
| Previous minor | ✅ (critical fixes only) |
| Older | ❌ |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately to:

**Mehdi Mahdian** — [m.mahdian@gmail.com](mailto:m.mahdian@gmail.com)

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

I will acknowledge receipt within 48 hours and aim to release a fix within 14 days for critical issues.

## Scope

This security policy covers:
- `MonaServer2.Service` REST API and SignalR hub
- `MonaServer2.Desktop` application
- `obs-mona-live` OBS Studio plugin
- Binary update mechanism

The MonaServer2 binary itself (bundled but authored by MonaSolutions) should be reported to [MonaSolutions](https://github.com/MonaSolutions/MonaServer2/security).
